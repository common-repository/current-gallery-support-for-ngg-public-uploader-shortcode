== Description ==

**The plugin adds support for quickly uploading the current gallery via the NGG Public Uploader shortcode,      
simply by typing *"current"* for the gallery id.**     
*The "Current gallery support for NGG Public Uploader shortcode" plugin is useful if you are building     
a public gallery where people can bring their own contribution.*    

= Usage: =

Just type *"current"* for the gallery id: `[ngg_uploader id=current]`.

== Installation ==

1. Extract the zip file.
1. Upload the folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
